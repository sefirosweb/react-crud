# react-crud

This project tries to simplify the process of creating, editing and deleting information,

In the vast majority of cases it can be simplified with a table and 3 buttons to do these operations.

You can directly use the `Crud` core or any component derived from it

## Installation

`npm install --save-dev @sefirosweb/react-crud`
